Este é um site produzido a partir dos requisitos feitos pelo professor Fabrício para a prova, P1.

criado somente com HTML e CSS o exercício demonstra os usos de encaminhamentos internos de página, imagens, display grid, flex e de formularios.

a pagina foi criada com a premissa de uma empresa de empregabilidade que usaria inteligencia artificial e um banco de dados amplo para selecionar os melhores perfis para as melhores vagas
